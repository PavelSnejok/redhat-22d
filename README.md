## Usage
### Prerequisites
   - Install the AWS CLI:
  ~~~bash
  pip install awscli
  ~~~
   - Install kubectl:
  ~~~bash
  brew install kubernetes-cli
  ~~~

## Run
 - use aws credantials from [AWS-312school](https://312school.awsapps.com/start#/)
 - run Makefile 
    ~~~bash
    make config
    ~~~# redhat-22d
